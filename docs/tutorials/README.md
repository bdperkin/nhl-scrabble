# Tutorials

Step-by-step lessons to help you learn NHL Scrabble from scratch.

## What are tutorials?

Tutorials are **learning-oriented** lessons that take you by the hand through a series of steps to complete a project. They are designed for beginners who want to get started with NHL Scrabble.

## Available Tutorials

1. **[Getting Started](01-getting-started.md)** - Your first NHL Scrabble analysis
   - Install the tool
   - Run your first analysis
   - Understand the basic output
   - Customize basic settings
   - *Time: ~15 minutes*

2. **[Understanding Output](02-understanding-output.md)** - Deep dive into reports
   - Conference standings
   - Division standings
   - Playoff brackets
   - Team details
   - Player scores
   - *Time: ~20 minutes*

3. **[First Contribution](03-first-contribution.md)** - Make your first code contribution
   - Set up development environment
   - Run tests
   - Make a small change
   - Submit a pull request
   - *Time: ~30 minutes*

## How to use these tutorials

- **Start at the beginning**: Tutorials build on each other
- **Follow every step**: Don't skip steps, even if they seem obvious
- **Type the commands**: Don't copy-paste - typing helps you learn
- **Experiment**: After completing a tutorial, try variations
- **Ask for help**: If you get stuck, see our [Support](../../SUPPORT.md) guide

## What next?

After completing these tutorials:

- **Solve specific problems**: Check out [How-to Guides](../how-to/)
- **Look up details**: Browse the [Reference](../reference/) documentation
- **Understand the system**: Read [Explanations](../explanation/) about architecture and design

## Prerequisites

All tutorials assume you have:

- **Python 3.10 or higher** installed on your system
- **Basic command-line knowledge** (navigating directories, running commands)
- **Internet connection** to fetch NHL roster data
- **Text editor** for viewing output files
