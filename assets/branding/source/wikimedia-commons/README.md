# Wikimedia Commons Reference Files

This directory contains SVG source files downloaded from [Wikimedia Commons](https://commons.wikimedia.org/) for use as reference material in creating the NHL Scrabble project logo.

## Purpose

These files are **reference materials** used to create derivative works or inspire design elements for the NHL Scrabble logo. They are **not** the final logo itself, but rather components, inspiration, or source material.

## Licensing & Attribution

All files in this directory are sourced from Wikimedia Commons and are licensed under open licenses that permit reuse. Proper attribution is provided in `ATTRIBUTION.md`.

### Reusing Content Outside Wikimedia

- **Policy**: https://commons.wikimedia.org/wiki/Commons:Reusing_content_outside_Wikimedia
- **Summary**: Content on Wikimedia Commons can be reused freely, provided you comply with the specific license terms (attribution, share-alike, etc.)

Files in this directory may be licensed under one or more of:

## Attribution Requirements

See `ATTRIBUTION.md` for detailed attribution for each file.

### General Attribution Format

For each file, provide:

- ✅ **Filename** - Name of the file
- ✅ **Source URL** - Original Wikimedia Commons page
- ✅ **Author** - Original creator(s)
- ✅ **License** - Specific license(s)
- ✅ **Modifications** - Any changes made to the original

## Adding New Files

When downloading new files from Wikimedia Commons:

1. **Download the SVG** from the Commons page
1. **Save to appropriate subdirectory** (hockey/, scrabble/, etc.)
1. **Update ATTRIBUTION.md** with complete attribution details (see template in ATTRIBUTION.md)
1. **Verify license compatibility** with MIT project license
1. **Commit with message**: `docs(branding): Add Wikimedia Commons reference file - [filename]`
